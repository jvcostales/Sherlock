Sherlock Guide
Sherlock uses Google Forms and Google Sheets to display data. Below are the steps to maximize your use.
Sign up and login
  1. If you don’t have an account yet, sign up using a strong combination of characters for your username and password.
  2. Login using the sign-up credentials you entered previously.
Set up your Google Forms and Google Sheets
  1. Set up your Google Forms here.
  2. Ensure that your form consists of the following (in order): Name, Address, Item, Amount, Date, Proof of Payment.
  3. If you have any other details needed from your customers, you can put them after the required details.
  4. In the responses tab, click the “Link to Sheets” button and confirm.
Set up your income page
  1. On the income page, click the “Select Sheets” button and enter your sheet's spreadsheet ID and range name.
  2. The spreadsheet ID is the last string of characters in the URL for your spreadsheet. For example, in this sample spreadsheet, the spreadsheet ID is 1WP_X46C7ylMbwD7WLOlx0EgUADf50UdMRIDCfiT9yNg.
  3. The range name is a syntax that contains the sheet name plus the starting and ending cell coordinates. For example, Sheet1!A:B refers to the first two columns of Sheet1.
  4. For more information, please visit Google Sheets API Overview.
  5. You can select a filter option on how you want to view your income database. This can be for the last 7 days, 28 days, or lifetime.
Set up your expenses page
  1. On the expenses page, click the “Add Expense” button and enter the necessary information: material, quantity, price, and date of expense.
  2. Ensure that all fields are duly filled out, as the system will show an error when invalid entries are entered.
  3. Clicking the “Submit” button will reflect the expense information on the page. In case of any mistake, you can use the delete button.
  4. You can select a filter option on how you want to view your expenses database. This can be for the last 7 days, 28 days, or lifetime.
